Setup Instructions

Follow these steps to create an alert in TradingView:

1
Open TradingView Chart

Go to TradingView and open the chart with your Pine Script indicator/strategy.

2
Create Alert

Click the Alert button (bell icon) at the top right, or press Alt + A.

3
Configure Alert Condition

Select your indicator/strategy and the condition that triggers the alert (e.g., "Any alert() function call").

4
Set Alert Name

Give your alert a descriptive name (e.g., "ZenithEdge EURUSD Signals").

5
Paste Webhook URL

In the Webhook URL field, paste your unique webhook URL from Step 1.

6
Paste JSON Message

In the Message field, paste the JSON template from Step 2. Make sure it's valid JSON!

7
Save & Test

Click Create and wait for your indicator to generate a signal. Check your ZenithEdge dashboard to verify it arrives!

Need Help?

Common Issues:

  • Signals not arriving? Check that your webhook URL is correct and your TradingView alert is active.
  • Invalid JSON error? Make sure your Pine Script has all required plot() outputs (plot_0 through plot_5).
  • Wrong data? Verify that {{ticker}}, {{interval}}, and plot values match your strategy outputs.
  • Webhook disabled? Use the "Toggle Status" button above to activate it.

Still stuck? Ask ZenBot in the dashboard chat for help!
